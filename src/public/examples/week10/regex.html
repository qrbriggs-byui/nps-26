Regular Expressions

Week 10 • Pattern matching with regex

Interactive Regex Tests

Regular expressions (regex) are patterns used to match and validate text. Try entering text in each field below to see how different patterns work.

Numbers Only

Pattern: /^\d+$/

Matches one or more digits with no other characters.

ZIP Code

Pattern: /^\d{5}(-\d{4})?$/

Matches 5 digits or 5 digits with optional 4-digit extension: 12345 or 12345-6789

US Phone Number

Pattern: /^\d{3}-\d{3}-\d{4}$/

Matches format: 123-456-7890 (3 digits, 3 digits, 4 digits with dashes)

Date (YYYY-MM-DD)

Pattern: /^\d{4}-\d{2}-\d{2}$/

Matches dates in ISO format: 2026-06-23

Hex Color Code

Pattern: /^#([0-9A-F]{3}){1,2}$/i

Matches hex color codes: #FFF or #FFFFFF (case-insensitive)

Username

Pattern: /^[a-zA-Z0-9_-]{3,16}$/

Letters, numbers, underscores, or dashes. Between 3 and 16 characters long.

Email Address

Pattern: /^[^\s@]+@[^\s@]+\.[^\s@]+$/

Matches a basic email format: something@domain.extension

Website URL

Pattern: /^https?:\/\/.+\..+$/

Matches HTTP or HTTPS URLs: http(s)://something.something

Strong Password

Pattern: /^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[@$!%*?&])[A-Za-z\d@$!%*?&]{8,}$/

At least 8 characters with uppercase, lowercase, number, and special character.

Regex Symbols & Operators Glossary

Symbol Name Description
^ Caret / Start anchor Matches the start of a string
$ Dollar / End anchor Matches the end of a string
. Dot / Any character Matches any single character except newline
* Asterisk / Zero or more Matches 0 or more of the preceding character
+ Plus / One or more Matches 1 or more of the preceding character
? Question mark / Optional Matches 0 or 1 of the preceding character
{n} Exact count Matches exactly n of the preceding character
{n,m} Range count Matches between n and m of the preceding character
[abc] Character class Matches any one character from the set: a, b, or c
[a-z] Character range Matches any character in the range: a through z
[^abc] Negated class Matches any character NOT in the set: a, b, or c
\d Digit Matches any digit (0-9). Same as [0-9]
\D Non-digit Matches any non-digit character
\w Word character Matches letters, digits, and underscore [a-zA-Z0-9_]
\W Non-word character Matches any non-word character
\s Whitespace Matches any whitespace character (space, tab, newline)
\S Non-whitespace Matches any non-whitespace character
| Alternation / OR Matches either pattern: cat|dog matches "cat" or "dog"
() Grouping Groups patterns together: (ab)+ matches "ab", "abab", etc.
(?=...) Positive lookahead Asserts that a pattern follows (without consuming characters)
\ Escape character Treats the next character as literal: \. matches a period
i flag Case-insensitive Makes the pattern match uppercase and lowercase: /hello/i
