Why this sizes value? The image is near full width on small screens, scales down at medium widths, and effectively caps at about 960px on large screens.

How to tell which image version is showing

  1. Read the Current resource line above. It shows img.currentSrc, the exact URL the browser chose.
  2. Open DevTools (F12) → Network tab → filter by Img.
  3. Resize the browser width, then refresh. Watch the requested image dimensions change (for example: 480/270, 640/360, 960/540).
  4. In Elements, select the image and confirm currentSrc in Properties/Console.
