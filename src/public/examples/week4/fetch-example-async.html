Loading Local Data with Fetch – Async/Await

1) Load a Local JSON File

This example loads ./data/parks.json using async/await.

async function loadJSON() {
  const response = await fetch("./data/parks.json");
  const data = await response.json();
  console.log(data);
}

2) Load a Local Text File

This example loads ./data/message.txt as plain text using async/await.

async function loadText() {
  const response = await fetch("./data/message.txt");
  const text = await response.text();
  console.log(text);
}

3) View the File in the Network Tab

  • Open the page in your browser.
  • Open Developer Tools.
  • Select the Network tab.
  • Refresh the page.
  • Click parks.json to view the JSON response.
  • Click message.txt to view the text response.
  • Use the Response tab to see the file contents.
