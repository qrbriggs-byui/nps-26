JSON Object Example

1) JSON Object

{
  "name": "Avery",
  "age": 20,
  "isStudent": true,
  "gpa": 3.85,
  "skills": ["HTML", "CSS", "JavaScript"],
  "address": {
    "city": "Rexburg",
    "state": "Idaho"
  },
  "middleName": null
}

2) How JSON Data Types Are Represented

  • String → "Avery"
  • Number → 20 or 3.85
  • Boolean → true or false
  • Array → ["HTML", "CSS", "JavaScript"]
  • Object → { "city": "Rexburg", "state": "Idaho" }
  • Null → null

3) Create a JSON Object from a JavaScript Literal

const student = {
  name: "Avery",
  age: 20,
  isStudent: true,
  gpa: 3.85,
  skills: ["HTML", "CSS", "JavaScript"],
  address: {
    city: "Rexburg",
    state: "Idaho"
  },
  middleName: null
};

This is a JavaScript object literal. It uses the same structure as JSON, but it is written in JavaScript.

4) Access Data in a JSON Object

student.name
student.age
student.isStudent
student.skills[0]
student.address.city
