JSON Scavenger Hunt Exercise

Use the person object in the script below and log the requested values to the JavaScript console.

How to open DevTools

  1. Open this page in your browser.
  2. Press F12 or Ctrl + Shift + I on Windows.
  3. Click the Console tab.

How to log values

Type a console.log() statement in the dev tools console, then press Enter and check the Console.

console.log(person.email);

Log the following:

  • Person's E-mail Address
  • Person's Full Name (First Middle and Last)
  • Person's Job Department
  • Person's Job Skills - First one listed
  • Person's Favorite Color
  • Person's Graduation Year from BYU-Idaho
  • Person's Apartment Coordinates - Latitude

Use dot notation and array indexing to access the data.
